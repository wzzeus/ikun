## 快速验证工具

```javascript
// ASCII 转字符
String.fromCharCode(72, 101, 108, 108, 111) // "Hello"

// 二进制转十进制
parseInt('01001000', 2) // 72

// URL 解码
decodeURIComponent('%48%65%6C%6C%6F') // "Hello"

// Base64 解码
atob('Q29kZQ==') // "Code"

// 十进制转十六进制
(3735928559).toString(16) // "deadbeef"

// Unicode 编码
"美".charCodeAt(0).toString(16) // "7f8e"
```
